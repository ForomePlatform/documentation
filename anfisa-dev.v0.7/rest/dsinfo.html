dsinfo¶

Dataset information

Synopsis¶

dsinfo

Arguments:

ds: dataset name

note: optional modifier text of note

Return value: Dataset Descriptor structure

Description¶

The request returns information about dataset in form of Dataset Descriptor structure.

If note argument is set, request modifies value of dataset note.
