adm_update¶

Force vault state update

Synopsis¶

adm_update

Arguments: none

Return value: "Updated" as plain string

Description¶

Initiates checking procedure for all datasets in the vault immediately. Does not wait for end of the procedure.

Comment¶

Standard procedure of the vault check starts on server in background periodically (once per 30 seconds by default service configuration) Using this call, one can initiate the check procedure immediately.
