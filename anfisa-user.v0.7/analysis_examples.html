Analysis examples¶

This section contains several examples of step-by-step instructions for analysis of variations.

Variant curation using built-in filtering options¶

  1. Contact administrator to upload the VCF file with the sequencing results

  2. Open Decision Tree panel, select and apply an appropriate built-in decision tree

  3. Create, name and save a new workspace contained variants satisfying the decision tree filtering

  4. Open created workspace from the workspace directory using link to the workspace

  5. Apply one of built-in filters from FILTERS drop-down menu

  6. Review, tag and create the user curation notes variants in Variant Information field in the FOROME user interface

  7. Export filtered and curated variants as a Microsoft EXCEL file.

Variant curation using custom filtering options¶

  1. Contact administrator to upload the VCF file with the sequencing results

  2. Open XL dataset

  3. Select and set filter options. Name and save custom decision filter

  4. Create, name and save new workspace containing variants satisfying the filtering

  5. Open created workspace from the workspace directory using link to the workspace

  6. Open FILTERS window. Select and set filter options. Name and save custom filter

  7. Apply custom filter

  8. Review, tag and create the user curation notes variants in Variant Information field on the FOROME user interface

  9. Export filtered and curated variants as EXCEL files.
